# KPengDeployTool

| 中文版 | English Version |
|--------|----------------|
| **KPengDeployTool** 是一款综合的移动应用打包和分发工具。它支持打包 Flutter、Unity、Cocos 和 UniApp 项目，内置内外网穿透功能，提供基于二维码的下载和分发，兼容 iOS 和安卓平台。 | **KPengDeployTool** is a comprehensive packaging and distribution tool for mobile applications. It supports packaging Flutter, Unity, Cocos, and UniApp projects with built-in internal/external network tunneling capabilities, providing QR code-based download and distribution for both iOS and Android platforms. |
