# KPengDeployTool

**KPengDeployTool** is a comprehensive packaging and distribution tool for mobile applications. It supports packaging Flutter, Unity, Cocos, and UniApp projects with built-in internal/external network tunneling capabilities, providing QR code-based download and distribution for both iOS and Android platforms.

## Version
Current version: **v2.1.0**

## Description
KPengDeployTool is a macOS application that simplifies the build and distribution process for cross-platform mobile applications. The tool supports multiple popular development frameworks and provides seamless internal/external network tunneling for easy app distribution via QR codes.

## Supported Frameworks
- Flutter
- Unity
- Cocos
- UniApp

## Features
- Multi-framework support (Flutter, Unity, Cocos, UniApp)
- Cross-platform compatibility (iOS and Android)
- Built-in internal/external network tunneling
- QR code-based download and distribution
- macOS native application
- Easy-to-use interface

## System Requirements
- macOS operating system
- Sufficient disk space for installation and operation
- Access to Flutter, Unity, Cocos, or UniApp project files

## Installation
1. Download the `.dmg` file from the releases section
2. Open the DMG file and follow the installation instructions
3. Launch the application from your Applications folder

## Usage
1. Open KPengDeployTool
2. Select your project framework (Flutter, Unity, Cocos, or UniApp)
3. Load your project files
4. Configure build settings
5. Initiate the packaging process
6. Use the built-in tunneling feature to generate QR codes for distribution
